Brick Viewer
============

This repository contains the source code of the Brick Viewer.

Usage
-----

You should be able to start brickv from source with "python main.py" in
the src/brickv/ directory.

The following libraries are required:
 * PyQt4
 * PyQwt5
 * pylab
 * scipy
 * pyopengl
 * numpy
 * matplotlib
